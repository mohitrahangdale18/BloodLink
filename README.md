# 🩸 BloodLink - Community Blood Donor Network

A full-stack MERN application connecting blood donors with people in need.

## 🚀 Live Demo
- Frontend: [your-vercel-link]
- Backend API: [your-render-link]

## ⚙️ Tech Stack
- MongoDB + Mongoose
- Express.js
- React (Vite)
- Node.js
- JWT Authentication
- bcrypt

## ✨ Features
- Register & Login with JWT auth
- Create donor profile
- Search compatible blood donors
- Emergency blood requests
- Real-time availability toggle

## 🛠️ Run Locally
```bash
# Backend
cd server
npm install
npm run dev

# Frontend
cd client
npm install
npm run dev
```
